A web page is a single document or file found on the World Wide Web, viewable through a web browser, and hosted on a web server. It's composed of content like text, images, videos, and interactive elements, often written in languages like HTML and styled with CSS. Web pages are connected to each other via hyperlinks, allowing users to navigate between them. A collection of related web pages linked under a common domain name constitutes a website. Key characteristics: Content: Web pages can contain text, images, videos, and other multimedia content. Structure: They are built using Hypertext Markup Language (HTML) to define the content and structure. Interactivity: Cascading Style Sheets (CSS) control the appearance, while JavaScript adds dynamic and interactive features. Navigation: Hyperlinks are used to connect different web pages, allowing users to move seamlessly between them. Unique Address: Each web page has a unique Uniform Resource Locator (URL) that allows users to access it. Static vs. Dynamic: Pages can be static, with unchanging content, or dynamic, displaying content that changes or is specific to the user. How it works: Request: When you enter a URL or click a link, your web browser sends a request to a web server. Serving: The web server retrieves the requested web page (an HTML document, for example) and sends it back to your browser. Display: Your web browser then renders the web page, displaying the content and interactive elements to you. Example: Think of a website as a book, and each web page as a single page within that book. The entire book forms a website, while each individual page within it is a web page.
