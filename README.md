# CampusCart

CampusCart is a polished front-end grocery planner designed as a stronger portfolio project for internship applications.

## What improved

- Rebuilt the project into a clean structure with `index.html`, `styles.css`, and `script.js`
- Added a more modern responsive layout and visual system
- Added search, category filtering, quantity controls, and budget tracking
- Added persistent cart state with `localStorage`
- Added a more complete checkout summary and confirmation flow
- Moved older practice files into the `archive/` folder so the repo root stays clean

## Project structure

```text
.
|-- archive/
|-- index.html
|-- styles.css
|-- script.js
`-- README.md
```

## How to run

Open `index.html` in a browser.

## Push to GitHub

Once Git is installed on your computer, you can run:

```bash
git init
git add .
git commit -m "Create CampusCart portfolio project"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/campuscart.git
git push -u origin main
```

If you want, the next step can be polishing the content even more with your name, internship focus, and a custom GitHub Pages deployment setup.
